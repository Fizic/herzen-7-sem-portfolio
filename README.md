# Портфолио - Кирилл Фирсов

Сайт портфолио на Vue.js и Nuxt 3.

## Технологии

- Nuxt 3
- Vue 3
- Tailwind CSS

## Установка

```bash
npm install
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000)

## Сборка для production

```bash
npm run build
```

## Структура проекта

- `pages/` - страницы приложения
  - `index.vue` - страница "Обо мне"
  - `works.vue` - страница "Мои работы в вузе"
- `components/` - компоненты
  - `NavBar.vue` - навигационная панель
  - `FileTree.vue` - компонент для отображения дерева файлов
- `layouts/` - макеты страниц
  - `default.vue` - основной макет
- `works/` - папка с работами из вуза (добавляйте файлы через git)
- `server/api/` - серверные API endpoints
  - `works.get.ts` - получение списка файлов из папки works
  - `works/[...path].get.ts` - скачивание конкретного файла (catch-all маршрут для вложенных путей)

## Функционал

### Страница "Обо мне"
- Отображение информации о профессиональном опыте
- История работы в компаниях
- Список используемых технологий

### Страница "Мои работы в вузе"
- Автоматическое отображение файлов и папок из директории `works/`
- Древовидная структура файлов
- Скачивание файлов
- Отображение размера файлов
- Автоматическое обновление после деплоя

## Как добавить работы

1. Добавьте файлы и папки в директорию `works/` проекта
2. Закоммитьте изменения в git:
   ```bash
   git add works/
   git commit -m "Добавлены работы из вуза"
   git push
   ```
3. После деплоя файлы автоматически появятся на сайте

Пример структуры:
```
works/
  ├── Курсовая работа 1/
  │   ├── report.pdf
  │   └── code/
  │       └── main.py
  ├── Лабораторная работа 2/
  │   └── lab2.docx
  └── Диплом/
      ├── thesis.pdf
      └── presentation.pptx
```
